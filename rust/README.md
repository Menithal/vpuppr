# Vpuppr Rust Lib

A [GDExtension](https://docs.godotengine.org/en/stable/tutorials/scripting/gdextension/what_is_gdextension.html) made
with [godot-rust](https://github.com/godot-rust/gdext).

This is resurrected version of the original repository that got archived and abandoned and adapted to work with the Godot 4.5 and modern godot-rust on version 0.4.0.

You may find the original archived version [here](https://github.com/virtual-puppet-project/vpuppr).
